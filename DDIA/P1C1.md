# Надёжные, масштабируемые и поддерживаемые приложения
Большинство современных приложений сталкиваются с нагрузкой, связанной с интенсивностью поступления обрабатываемых данных, в противоположность нагрузке, вызванной затратами на вычислительные операции (далее - **высоконагруженные приложения**). Основные проблемы это количество данных, сложность их структуры, и скорость, с которой данные поступают.

Высоконагруженные приложения создаются из стандартных компонентов, предоставляющих необходимую функциональность:
- базы данных
- кэши - сохранение результатов ресурсозатратных операций
- поисковые индексы
- системы потоковой обработки - отсылка сообщений другим процессам для асинхронной обработки
- системы пакетной обработки - периодический приём больших объёмов агрегированных данных

Все вышеперечисленные системы можно собирательно назвать *системами обработки данных*.
